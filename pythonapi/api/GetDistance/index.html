    GetDistance

    1. GetDistance (cm)

    參數

    • Pin_Trig : 2~21
    • Pin_Echo : 2~21

    Sample 

    print('Distance='+str(v5012.GetDistance('A2','A3')))

    2. GetDistanceInTime (ms)

    參數

    • Pin_Trig : 2~21
    • Pin_Echo : 2~21

    Sample 

    print('Distance='+str(v5012.GetDistanceInTime('A2','A3')))
